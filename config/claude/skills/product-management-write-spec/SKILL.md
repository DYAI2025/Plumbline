# Product Management: Write Spec

Turn a vague product request into a concise feature spec.

## Output
- Problem statement
- Users/personas
- Jobs to be done
- Scope and non-goals
- User stories
- Success metrics
- Risks and dependencies
- Questions that must be answered before engineering starts

Hand the result to `ai-native-prd-architect` for engineering-grade REQ IDs and traceability.
