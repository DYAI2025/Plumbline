# Writing Skills

## Quality bar
- One capability per skill.
- Frontmatter includes `name` and a use-oriented `description`.
- Include trigger conditions, workflow, guardrails, and outputs.
- Prefer concise checklists over broad essays.
- Check for overlap with existing skills before creating a new one.
- Validate that examples do not encourage unsafe or unverified claims.
