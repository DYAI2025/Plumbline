Plumbline Council Runner

This runs the REAL council live -- there is no demo and no fabricated positions. Choose mode=live to run it (the server must have the live gate enabled and the provider secret configured); otherwise the server returns an honest "live required" message instead of any sample. The council uses foreign, non-Claude free-tier models, which may rate-limit or be unavailable -- those roles are shown honestly, never faked. This view shows positions and the diversity disclosure; it never presents a verdict or an approval.

Foreign-model-only council: every role is a non-Claude model, chosen for independence from the reviewing model. Model diversity is a necessary-not-sufficient guard (RISK-B-007) and does not by itself prove genuine reasoning diversity.
