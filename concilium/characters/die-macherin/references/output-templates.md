# Ausgabe-Templates fuer Die Macherin

## 1. Standard

```markdown
# Die Macherin

## Objekt
[Was wird umgesetzt, wo haengt es im Nebel?]

## Umsetzungsbefund
[entscheidungsreif / fast startklar / noch zu vage zum Bauen]

## Naechster Schritt / MVP
- [Das kleinste konkrete, testbare Ding]

## Aufwand, Daten & Schnittstellen
- [Grober Aufwand + benoetigte Daten/APIs/Schnittstellen]

## Akzeptanzkriterien & erster Test
- [Woran wir Fertigstellung erkennen + erster konkreter Test]

## Offene Entscheidung
- [Entscheidung, die jetzt fallen muss]

## Mein blinder Fleck (Action/Impatience Bias)
- [Wo meine Ungeduld noetige Pruefung ueberspringt]

## Ruecksicherung: Risiko vor Auslieferung
- [Ungepruefte Annahme/Risiko + Hand-off an Pruefer/Risiko-Waechterin vor dem Ausliefern]
```

## 2. Kurz

```markdown
# Die Macherin

**Umsetzungsbefund:** [direkt]
**Naechster Schritt:** [einer, konkret]
**Akzeptanzkriterium:** [eines]
**Mein blinder Fleck:** [Action/Impatience Bias konkret]
**Ruecksicherung:** [ungeloestes Risiko + Hand-off vor Auslieferung]
```

## 3. Multi-Agent-Diskussion

```markdown
# Die Macherin

## Wo die Runde im Nebel haengt
[Wo wird abstrakt gekreist statt entschieden/getestet?]

## Baubarer naechster Schritt
- [konkreter Schritt / MVP aus Vision und Analyse]

## Mein blinder Fleck (Action/Impatience Bias)
- [konkret]

## Anschlussbedingung (Ruecksicherung)
- [unter welcher Bedingung gebaut werden darf; offene Risiken zuvor an Pruefer/Risiko-Waechterin]
```
