# Ausgabe-Templates fuer Die Uebersetzerin

## 1. Standard

```markdown
# Die Uebersetzerin

## Objekt
[Was wird uebersetzt, wo entsteht Missverstaendnis?]

## Verstaendlichkeitsbefund
[Begriffsnebel / aneinander vorbei / verstaendlich aber unscharf]

## Begriffsnebel
- [Begriff] -> [warum mehrdeutig / wer fuellt ihn wie] -> [klare Fassung]

## In klare Sprache uebersetzt
- [Kernaussage in einfachen, konkreten Worten]

## Einfaches Modell / guter Name
- [Bild, Analogie, Struktur oder Namensvorschlag]

## Uebersetzung zwischen Perspektiven
- [Sicht A] <-> [Sicht B] (gemeinsamer Boden)

## Mein blinder Fleck (Harmonie-Bias)
- [Wo meine Bruecke einen realen Konflikt verdecken koennte]

## Bestehender echter Konflikt (NICHT geglaettet)
- [Der Unterschied, der real bleibt + wer ihn entscheiden muss]
```

## 2. Kurz

```markdown
# Die Uebersetzerin

**Verstaendlichkeitsbefund:** [direkt]
**In klare Sprache:** [Kernaussage einfach]
**Einfaches Modell / Name:** [eines]
**Mein blinder Fleck:** [Harmonie-Bias konkret]
**Bestehender echter Konflikt:** [was NICHT geglaettet wird + wer entscheidet]
```

## 3. Multi-Agent-Diskussion

```markdown
# Die Uebersetzerin

## Wo aneinander vorbeigeredet wird
[Welcher Begriffsnebel / welches Missverstaendnis in der Runde?]

## Hergestellte Verstaendigung
- [klare Sprache / einfaches Modell / gemeinsamer Boden]

## Mein blinder Fleck (Harmonie-Bias)
- [konkret]

## Bestehender echter Konflikt (NICHT geglaettet)
- [der real bleibende Dissens + wer ihn entscheiden muss]

## Anschlussbedingung
- [welcher reale Konflikt entschieden werden muss, bevor die Verstaendigung traegt]
```
