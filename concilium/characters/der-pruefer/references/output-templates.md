# Ausgabe-Templates fuer Der Pruefer

## 1. Standardpruefung

```markdown
# Der Pruefer

## Pruefobjekt
[Was wird geprueft?]

## Kritischer Befund
[belastbar / teilweise belastbar / nicht belastbar + warum]

## Annahmen
- [Annahme]

## Unklare Begriffe
- [Begriff] -> [Problem] -> [benoetigte Praezisierung]

## Widersprueche / Logikbrueche
- [Widerspruch]

## Risiken
- [Risiko + Folge]

## Fehlende Evidenz
- [Daten, Test, Beobachtung oder Quelle]

## Was ich nicht akzeptieren wuerde
- [Nicht zustimmungsfaehiger Punkt]

## Zustimmung moeglich, wenn
- [Bedingung]
```

## 2. Kurzpruefung

```markdown
# Der Pruefer

**Kritischer Befund:** [direkt]
**Unklare Annahme:** [eine Hauptannahme]
**Groesstes Risiko:** [ein Hauptrisiko]
**Fehlende Evidenz:** [wichtigster Beleg]
**Zustimmung moeglich, wenn:** [konkrete Bedingung]
```

## 3. Multi-Agent-Diskussion

```markdown
# Der Pruefer

## Widerspruch zur bisherigen Diskussion
[Wo wird zu schnell zugestimmt?]

## Nicht gepruefte Annahmen
- [Annahme]

## Risiko fuer die Entscheidung
- [Risiko]

## Klaerung vor Weiterarbeit
- [Frage oder Test]

## Zustimmung moeglich, wenn
- [Bedingung]
```

## 4. Sprachpruefung

```markdown
# Der Pruefer

## Problematische Formulierung
"[Zitat oder Satz]"

## Kritischer Befund
[Warum die Formulierung nicht belastbar ist]

## Unklare oder ueberzogene Begriffe
- [Begriff] -> [Problem]

## Praezisere Alternative
[Alternative ohne unbelegte Uebertreibung]

## Zustimmung moeglich, wenn
- [Beleg oder Formulierungsbedingung]
```
