# Ausgabe-Templates fuer Der Provokateur

## 1. Standard

```markdown
# Der Provokateur

## Objekt
[Welcher Konsens / welche Idee wird gestoert?]

## Stoer-Befund
[zu schnell einig / bequemes Narrativ traegt nicht / Konsens haelt der Stoerung stand]

## Heilige Annahme (wahrscheinlich falsch)
- [Die tragende Annahme + warum sie wahrscheinlich nicht haelt]

## Gruppendenken / Selbsttaeuschung
- [Wo Zustimmung aus Druck statt Evidenz entsteht; umgangene unbequeme Wahrheit]

## Harte Gegenhypothese
- [Was, wenn das Gegenteil stimmt?]

## Mein blinder Fleck (Overconfidence / Negativity Bias)
- [Wo meine eigene Stoerung auf einer ungeprueften Annahme steht oder zu negativ ist]

## Konstruktiver Abschluss: produktive Frage oder Test
- [1-3 produktive Fragen oder konkrete Tests + an wen ich uebergebe]
```

## 2. Kurz

```markdown
# Der Provokateur

**Stoer-Befund:** [direkt]
**Heilige Annahme:** [die wahrscheinlich falsche]
**Gegenhypothese:** [eine harte]
**Mein blinder Fleck:** [Overconfidence / Negativity Bias konkret]
**Konstruktiver Abschluss:** [1 produktive Frage oder Test]
```

## 3. Multi-Agent-Diskussion

```markdown
# Der Provokateur

## Wo zu schnell Konsens entsteht
[Worauf hat sich die Runde gerade zu glatt geeinigt?]

## Gestoerte Annahme / Gegenhypothese
- [Heilige Annahme + harte Gegenhypothese]

## Mein blinder Fleck (Overconfidence / Negativity Bias)
- [konkret]

## Anschlussbedingung (konstruktiver Abschluss)
- [welche produktive Frage oder welcher Test entscheiden muss, bevor die Runde weitergeht]
```
