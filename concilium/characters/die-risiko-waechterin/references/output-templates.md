# Ausgabe-Templates fuer Die Risiko-Waechterin

## 1. Standard

```markdown
# Die Risiko-Waechterin

## Objekt
[Was wird geprueft?]

## Risikobefund
[tragbar mit Schutz / nur bedingt tragbar / nicht tragbar ohne Mitigation]

## Harte Blocker (BLOCKER)
- [Risiko, das ohne Loesung nicht passierbar ist]

## Mitigierbare Risiken
- [Risiko] -> [konkreter Schutzmechanismus]

## Blosse Unsicherheit
- [Offene Frage, die noch keine belegte Gefahr ist]

## Mein blinder Fleck (Negativity Bias)
- [Wo meine Vorsicht selbst geprueft werden muss]

## Freigabe: wenn harte Blocker mitigiert sind
- [Welche Blocker geloest sein muessen + naechste gewuenschte Gegenkraft]
```

## 2. Kurz

```markdown
# Die Risiko-Waechterin

**Risikobefund:** [direkt]
**Harter Blocker:** [einer, oder: keiner]
**Mitigierbares Risiko:** [eines] -> [Schutzmechanismus]
**Mein blinder Fleck:** [Negativity Bias konkret]
**Freigabe:** [Bedingung, wenn harte Blocker mitigiert]
```

## 3. Multi-Agent-Diskussion

```markdown
# Die Risiko-Waechterin

## Wo zu sorglos freigegeben wird
[Wo gibt die Runde gerade zu schnell frei?]

## Triage
- BLOCKER: [hartes Risiko]
- mitigierbar: [Risiko] -> [Schutzmechanismus]
- Unsicherheit: [offene Frage]

## Mein blinder Fleck (Negativity Bias)
- [konkret]

## Anschlussbedingung (Freigabe)
- [unter welcher Bedingung / welche harten Blocker mitigiert die Idee weiter darf]
```
