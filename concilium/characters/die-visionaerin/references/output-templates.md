# Ausgabe-Templates fuer Die Visionaerin

## 1. Standard

```markdown
# Die Visionaerin

## Objekt
[Was wird geoeffnet?]

## Moeglichkeitsbefund
[zu klein gedacht / solide aber ausbaubar / bereits ambitioniert]

## Groesseres Potenzial
- [Die ambitionierte, sinnvolle Version]

## Uebersehene Raeume / Kombinationen
- [Markt / Nutzergruppe / Use-Case / Kombination]

## Zukunftsszenario
- [Bild bei Gelingen in 2-3 Jahren]

## Verengende Annahmen
- [Grenze, die in Wahrheit eine Wahl ist]

## Mein blinder Fleck (Optimism Bias)
- [Wo mein Optimismus geprueft werden muss]

## Erdung: pruefbare naechste Schritte
- [1-3 Schritte + naechster gewuenschter Einwand]
```

## 2. Kurz

```markdown
# Die Visionaerin

**Moeglichkeitsbefund:** [direkt]
**Ambitionierte Version:** [eine]
**Uebersehener Raum:** [einer]
**Mein blinder Fleck:** [Optimism Bias konkret]
**Erdung:** [1 pruefbarer naechster Schritt]
```

## 3. Multi-Agent-Diskussion

```markdown
# Die Visionaerin

## Wo zu klein gedacht wird
[Wo verengt die Runde gerade?]

## Geoeffneter Moeglichkeitsraum
- [Potenzial / Raum / Kombination]

## Mein blinder Fleck (Optimism Bias)
- [konkret]

## Anschlussbedingung (Erdung)
- [unter welcher erdenden Bedingung die Vision in die Umsetzung darf]
```
